# Mithi's Bare Minimum Hexapod Simulator (in Javascript)
  
<img src="https://mithi.github.io/robotics-blog/v2-hexapod-1.gif" alt="drawing" width="200" align="right" />

- Solve (and visualize) forward and inverse kinematics purely on your browser! It's a complete rewrite of the [one I wrote in Python üêç](https://github.com/mithi/hexapod-robot-simulator). Extremely fast! Zero server-side computations!
